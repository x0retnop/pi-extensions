 Фидбэк по удобству работы с браузером

 Что хорошо:
 - CDP-подключение к уже запущенному Chrome работает стабильно — не нужно открывать новый браузер каждый раз, сессия с cookies
   сохранена.
 - snapshot даёт отличную картину доступных элементов с ref-ссылками.
 - eval позволяет выполнять произвольный JS — очень гибко для кастомных селекторов.
 - tabs + cdp_url — удобно ориентироваться между вкладками.

 Что требует внимания:
 1. ref-ссылки из snapshot не всегда работают напрямую. click по [ref="e93"] не сработал — пришлось искать через eval/aria-label. Это
    может быть связано с динамическим рендерингом React (Grok использует React, и ref могут меняться при ре-рендере).
 2. Нет прямой кнопки отправки — пришлось искать её через eval. Кнопка появляется динамически после ввода текста.
 3. Нет нативного press(Enter) — было бы удобнее для отправки формы, чем искать кнопку. type не поддерживает отправку по Enter
    (возможно, стоило попробовать browser.type с \n).
 4. wait по ms — единственный вариант когда нет надёжного селектора для ожидания загрузки ответа. Хорошо бы иметь wait по тексту или
    по отсутствию индикатора загрузки.

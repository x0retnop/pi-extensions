Что с "open code бесплатно"
Пользователь имеет в виду OpenCode (opencode.ai) — это open-source терминалный AI-кодинг агент (типа Cursor в терминале, от SST). Там сейчас доступна MiniMax M3 Free (или minimax-m3-free).
Как это работает бесплатно:

Через их сервис OpenCode Zen (или Go) они хостят урезанную версию M3.
Бесплатный вариант: ~200k токенов контекста (вместо полного 512k–1M).
Можно попробовать прямо сейчас без оплаты (установка через curl или npm).
Полная версия (больший контекст + лучшие лимиты) — через платный Token Plan MiniMax или их подписку OpenCode Go.

Как попробовать:

Установи: curl -fsSL https://opencode.ai/install | bash (или npm i -g opencode-ai)
Запусти opencode и выбери MiniMax-провайдера (там есть бесплатный вариант).
Или через opencode auth login и выбери MiniMax.

Многие сейчас пишут, что именно так можно пощупать M3 бесплатно (с ограничениями).
Open-weights / open-source
MiniMax традиционно выпускает open-weights модели (M1, M2, M2.5 и т.д. лежат на Hugging Face под permissive лицензиями). Для M3 глава инженерии и аккаунты компании намекали «open source incoming with M3».
На момент сейчас (1 июня) веса ещё не выложены, но обещают скоро на Hugging Face. Пока модель доступна через их API + через платформы вроде OpenCode / OpenRouter.
Если хочешь максимально бесплатно и локально — жди выкладки весов (скорее всего MIT/Apache-подобная лицензия, как у предыдущих). Тогда можно будет запускать через vLLM, Ollama и т.д. на своём железе.

1. Самый простой и лучший вариант (через твою подписку OpenCode Go)
У тебя уже есть OpenCode Go — это отличный прокси-провайдер.
OpenCode Go даёт доступ к MiniMax M3 (модель minimax-m3) по очень щедрым лимитам за $10/мес.
Как настроить в Pi CLI:
Pi CLI поддерживает OpenAI-совместимые провайдеры. Нужно добавить custom модель:
YAML# В конфиге Pi CLI (обычно ~/.pi/config.yaml или через команду config)
models:
  - name: minimax-m3
    provider: openai
    model: minimax-m3
    apiBase: https://opencode.ai/zen/go/v1
    apiKey: "твой_opencode_go_api_key"
    # Можно добавить:
    # contextWindow: 1000000
Или если Pi позволяет через UI/команду:

Добавь новый провайдер → OpenAI compatible
Base URL: https://opencode.ai/zen/go/v1
API Key: твой Go-ключ
Model name: minimax-m3

После этого в Pi можешь выбирать эту модель.
2. Бесплатный MiniMax M3 (ограниченный)
Если хочешь попробовать именно free tier без расхода Go-лимитов:

Используй OpenCode Zen free endpoint (там есть minimax-m3-free или похожий).
Base URL обычно: https://opencode.ai/zen/v1 или /zen/go/v1
Model: minimax-m3-free (или minimax-m3 с free планом)

Там контекст меньше и лимиты жёстче, но для тестов отлично.
3. Прямой MiniMax API (если хочешь максимум)
Если купишь Token Plan на minimax.io, то:

Base URL: https://api.minimax.io/v1 (или anthropic-совместимый)
Model: MiniMax-M3

Но через OpenCode Go сейчас выгоднее и проще.

Совет:
Сначала попробуй через OpenCode Go — у тебя уже есть подписка, так что M3 должен быть доступен сразу. Если в Pi CLI не подхватывается — кинь сюда кусок твоего конфига, подскажу точнее.
